Building FyelSrvr

Raw Sockets → a Secure Static Server

Six steps, one Ruby surface, compiled with Spinel · URUG Meetup

July 15th 2026

The idea

One web server, grown one file at a time

  • Start at the metal: a Socket we open by hand.
  • End with a real static file server: routing, MIME types, path-traversal defense, directory listings, file sizes.
  • Every step is a standalone program that compiles and runs under plain ruby.

The whole progression lives in source/fyel_srvr_1.rb … _6.rb.

The map

Six steps

  • 1 · Raw POSIX sockets (via FFI)
  • 2 · Minimal HTTP "Hello World" (TCPServer shim)
  • 3 · Request parsing + hard-coded routes
  • 4 · Static files + directory listings
  • 5 · Path sanitization + content types
  • 6 · FyelSrvr: CLI flags, parent links, icons, sizes, index.html

Each slide shows the real, shipped code.

Step 1 / 6

Raw sockets

Open a listening socket by hand

# fyel_srvr_1.rb
def start_server(port)
  server = Socket.new(Socket::AF_INET, Socket::SOCK_STREAM, 0)
  server.setsockopt(Socket::SOL_SOCKET, Socket::SO_REUSEADDR, 1)
  server.bind(Socket.sockaddr_in(port, "0.0.0.0"))
  server.listen(10)
  loop do
    accepted = server.accept
    client = accepted[0]
    respond_to_client(client)
  end
end

The full BSD-socket dance: create, setsockopt, bind, listen, accept.

Step 1 · the response

Hand-write one HTTP frame, then close

def respond_to_client(client)
  begin
    body = "<html><body><h1>Hello World from raw Socket!</h1></body></html>"
    response = "HTTP/1.1 200 OK\r\n" \
               "Content-Type: text/html\r\n" \
               "Content-Length: #{body.bytesize}\r\n" \
               "Connection: close\r\n" \
               "\r\n" \
               "#{body}"
    client.write(response)
  ensure
    client.close                # always close, even on error
  end
end
  • No framework, no parsing yet, just bytes on a wire.

Step 1 · the catch on Spinel

There is no require "socket"

$ spinel -E -e 'require "socket"; puts 1'
spinel: cannot load such file -- socket      # gone by design

The problem

Spinel ships no Ruby socket library, and no Socket class.

The fix

Bind the runtime's native sp_net_* primitives via FFI and wrap them in a Socket-shaped API, in lib/socket_shim.rb.

# compile step 1 with the project's C socket extension linked in:
$ spinel --link native/socket_ext/socket_ext.c source/fyel_srvr_1.rb -o fyel_srvr_1
Step 2 / 6

The shim

Same idea, a Ruby-shaped TCPServer

# fyel_srvr_2.rb
require_relative "lib/socket_shim"

def start_server(port)
  TCPServer.open("0.0.0.0", port) do |server|
    puts "Server listening natively."
    loop do
      respond_to_client(server.accept)
    end
  end
end

start_server(8080)

The socket boilerplate disappears behind TCPServer.open.

Step 2 · why this matters

One surface, two runtimes

Under CRuby

lib/socket_shim.rb just require "socket", you get the real stdlib TCPServer.

Under Spinel

The shim defines TCPServer over sp_net_* FFI, so the exact same fyel_srvr_2.rb compiles to a native binary.

  • From here on, the servers are pure application code, no sockets in sight.
  • Response body is still one hard-coded string. Next: read what the client asked for.
Step 3 / 6

Parse + route

Read the request line, pick a path

# fyel_srvr_3.rb
raw_request  = client.recv(2048)          # :binstr -> full String
lines        = raw_request.split("\r\n")
request_line = lines[0] || ""             # "GET /about HTTP/1.1"

parts  = request_line.split(" ")
method = parts[0] || "GET"
path   = parts[1] || "/"
  • recv(2048) returns a robust binary-safe Ruby String.
  • String splitting is all we need to isolate method and path.

Step 4 / 6

Serve real files

Map the URL path onto the filesystem

# fyel_srvr_4.rb
def handle_request(path)
  local_path = ".#{path}"                  # "/img.png" -> "./img.png"

  return not_found unless File.exist?(local_path)

  if File.directory?(local_path)
    body = build_directory_list(local_path, path)
    ["200 OK", "text/html", body]
  else
    body = File.read(local_path)           # binary-safe String
    ["200 OK", content_type_for(local_path), body]
  end
end
  • File.exist?, File.directory?, File.read, all backed by native stat/read.

Step 4 · directory index

Dir.entries → a clickable listing

def build_directory_list(dir_path, requested_path)
  base_url = requested_path.end_with?("/") ? requested_path : "#{requested_path}/"
  html = "<h1>Index of #{requested_path}</h1><hr><ul>"

  Dir.entries(dir_path).each do |entry|    # readdir() under the hood
    next if entry == "."
    html += "<li><a href=\"#{base_url}#{entry}\">#{entry}</a></li>"
  end

  html + "</ul><hr>"
end

A working file browser, and still one dependency-free file.

Step 5 / 6

Make it safe

Defuse ../ path traversal

# fyel_srvr_5.rb
def sanitize_path(raw_path)
  clean    = raw_path.gsub("\\", "/")
  segments = clean.split("/")
  safe_segments = []
  segments.each do |segment|
    next if segment == "" || segment == "."
    if segment == ".."
      safe_segments.pop          # climb back down, never above root
    else
      safe_segments.push(segment)
    end
  end
  "/" + safe_segments.join("/")
end
  • /../../etc/passwd collapses to a root-relative path, it can't escape.

Step 5 · content types + disposition

Serve web files inline, download the rest

if local_path.end_with?(".html") || local_path.end_with?(".htm")
  content_type = "text/html";  is_web_file = true
elsif local_path.end_with?(".css")
  content_type = "text/css";   is_web_file = true
elsif local_path.end_with?(".png")
  content_type = "image/png";  is_web_file = true
# ... js, jpg, svg, json, txt ...
end

disposition = is_web_file ? "inline"
                          : "attachment; filename=\"#{file_name}\""

Plus folder/file SVG icons in the listing, all inline ASCII.

Everything from steps 1–5, assembled into one program.

Step 6 · index.html handoff

Directory? Serve its index, else list it

# fyel_srvr_6.rb
if File.directory?(local_path)
  separator       = local_path.end_with?("/") ? "" : "/"
  index_html_path = "#{local_path}#{separator}index.html"

  if serve_index && File.exist?(index_html_path) && !File.directory?(index_html_path)
    return serve_file_payload(index_html_path)   # serve the page
  else
    body = build_directory_list(local_path, safe_url_path)
    return ["200 OK", "text/html", "inline", body]
  end
end

Step 6 · human-readable sizes

Integer math only, no formatting lib

def format_file_size(bytes)
  if bytes < 1024
    "#{bytes} B"
  elsif bytes < 1048576
    kb        = bytes / 1024
    remainder = (bytes % 1024) * 10 / 1024
    "#{kb}.#{remainder} KB"
  else
    mb        = bytes / 1048576
    remainder = (bytes % 1048576) * 10 / 1048576
    "#{mb}.#{remainder} MB"
  end
end

Shown in the listing next to each file via File.size.
